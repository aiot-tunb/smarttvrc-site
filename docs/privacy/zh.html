隐私政策

English · Tiếng Việt · 日本語 · 한국어 · 简体中文
最后更新：April 29, 2026

Remote SmartTV 是一款完全本地化的电视遥控应用。我们致力于保护您的隐私，并对应用的运作方式保持透明。

1我们收集的信息

Remote SmartTV 不会在外部服务器上收集、传输或存储任何个人数据。所有数据——包括电视连接信息、应用设置和 PIN 码——均本地存储在您的设备上，不会离开设备。

2信息使用方式

电视控制

电视 IP 地址和配对凭据本地存储用于建立连接，不会与任何第三方共享。

语音搜索

麦克风音频实时直接流式传输到电视，不会被录制、存储或传输到任何服务器。

应用锁

您的 PIN 码通过系统钥匙串安全存储在设备上。生物特征认证使用设备内置系统，应用无法访问生物特征数据。

3所需权限

  • 本地网络 — 需要此权限以发现并连接您 Wi-Fi 网络上的智能电视。
  • 麦克风 — 仅在语音搜索期间使用。音频直接流式传输到电视。
  • 面容 ID / 指纹 — 在启用应用锁时用于解锁应用。生物特征数据完全由设备操作系统管理。

4数据存储与安全

所有应用数据仅存储在您的设备上。敏感数据（PIN 码、配对密钥）使用系统钥匙串保存。没有数据会传输到我们的服务器。

5第三方服务

应用使用 Firebase Crashlytics 和 Firebase Analytics 接收匿名崩溃报告和使用统计数据。这些服务不收集个人可识别信息。

6儿童隐私

Remote SmartTV 不会有意收集 13 岁以下儿童的任何信息。应用不需要账户注册或任何个人数据。

7政策变更

我们可能会不时更新本隐私政策。更新版本将通过上方的"最后更新"日期标注。变更后继续使用应用即表示接受更新后的政策。

8联系我们

如对本隐私政策有任何疑问，请通过以下方式联系我们：support@i2r.org.com

9适用法律

本隐私政策受越南法律管辖。任何争议应提交越南法院专属管辖。
